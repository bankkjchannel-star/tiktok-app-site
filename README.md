# TikTok App Website (Template)

This is a minimal static website for:
- Home page
- Terms of Service
- Privacy Policy

## 1) Replace placeholders
In **index.html**, **terms.html**, **privacy.html** replace:
- APP_NAME
- CONTACT_EMAIL
- WEBSITE_URL

Tip: Use Find/Replace in your editor.

## 2) Publish via GitHub Pages (fast)
1. Create a new repo on GitHub (public is easiest)
2. Upload these files to the repo root:
   - index.html
   - terms.html
   - privacy.html
   - style.css
3. Go to Settings → Pages
4. Source: Deploy from a branch
5. Branch: main, folder: /(root)
6. Save → wait until you get a URL like:
   https://<your-username>.github.io/<repo-name>/

## 3) Use URLs in TikTok Developer Portal
Set:
- Web/Desktop URL: your GitHub Pages URL (home)
- Terms of Service URL: .../terms.html
- Privacy Policy URL: .../privacy.html
